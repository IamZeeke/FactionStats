# FactionStats

Paper 1.21 plugin for FactionsUUID servers.

## Features
- Real-time animated scoreboard
- Shows your faction stats
- Top 5 factions by power
- Crossed swords animation

## Requirements
- Paper 1.21.x server
- FactionsUUID 4.4.0+

## Usage
1. Admin clones repo
2. Build jar
3. Drop jar in plugins folder
4. Restart server
5. Players use /fstats
